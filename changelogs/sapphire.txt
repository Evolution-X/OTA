This is Android 14 with September 2024 QPR3 (r67) security patches

Notes:
==============================
- LineageOS-based source. Signed build.
- Play Integrity passes, RCS works.
- MiuiCamera Included
- Dolby Atmos Included

Device changes:
==============================
- Disable alpha compositing in WM
- Disable vsync for CPU rendered Apps
- Enable config_avoidGfxAccel
- Offload WM shell to another thread
- Make the UI smoother
- Remove ro.vendor.perf.scroll_opt
- Force disable iorapd
- Disable all logging
- Disable TDLS offchannel
- Smarter decisions on whether to use a 2 or 5Ghz AP
- Enable Optimized Power Management
- Disable RX wakelock feature
- Enable DFS channel scanning in P2P search
- Enable automatic channel selection
- Disable RescueParty
- Kill IMS logspam
- Add xiaomi perf changes
- Avoid delay for vblank disable from DRM
- Do not balance msm_drm and kgsl_3d0 IRQs
- Dynamically set IRQ affinity for KGSL and MSM_DRM
- Disable multiple kswapd threads
- Disable native stats collection service
- Compact cached app heaps in the background
- Allow more cached apps in the background
- Add SystemUI restart in restart advanced menu
- Configure zram to 50% of physical
- Tune khaje boosts values
- Boost GPU on specific events
- Enable performance mode on khaje
